# inflight-infotainment-system

A web-based in-flight infotainment system prototype.

## Project Overview

This app is intended to run in a web browser and provide passengers with access to media, flight information, food and beverage ordering, shopping, safety information, and accessibility-related features.

## Branch Naming Examples:

- `feature/media-catalog`
- `feature/flight-info`
- `fix/navbar-layout`

# Workflow Guidelines

1. Pull the latest version of `main`.
2. Create a new branch for your task.
3. Make your changes.
4. Commit with a clear message.
5. Push your branch.
6. Open a pull request into `main`.

## Rules

- Do not push directly to `main`.
- Resolve and review ALL merge conflicts before merging into `main`.
- Keep pull requests focused on one feature or fix.
